# Hi, I'm Jonathan 👋

**Full-stack developer** building web applications, internal automation systems, and AI-powered tools — from WordPress landing pages to production CRM frontends and office infrastructure dashboards.

## My path in IT

I started with **WordPress websites** while studying and working as a **system administrator** in parallel — and kept growing as a developer from day one.

Over the years my work expanded across several directions:

- **Web development** — dozens of business websites, WordPress themes & plugins, landing pages, and marketplace frontends
- **CRM & product UI** — Vue/Nuxt frontends connected to backend systems, instructor search platforms, account dashboards
- **Sysadmin & DevOps** — server setup, trackers, deployment, network and office infrastructure
- **Automation** — Telegram bots, lead pipelines, n8n workflows, internal tools that save teams hours every week
- **Office management systems** — unified dashboards for access control, user provisioning, and service integrations
- **Real estate tech** — XML-fed property catalogs, lead generation platforms, presentation tools
- **AI products** — news analysis & publishing bots, conversational agents that handle client requests like a live administrator

Today I combine all of this into **full-stack delivery**: I can own a project from server and database to UI and AI layer.
